Job Management via SCDS API

Get all Commands

The HTTP method to be used is:
URI will be /command
Parameter: group
Default: All groups



Get all Command States

The HTTP method to be used is:
URI will be /command/state
Parameters: none



Get all Command Steps

The HTTP method to be used is:
URI will be /command/step
Parameters: none



Run a Command

The HTTP method to be used is:
URI will be /command/{commandname}
Parameters: systemid, nodeid, username, parameters (command dependent)









Get One Job

The HTTP method to be used is:
URI will be /task/{taskid}
Parameters: taskid, state, group, nodeid, fields







Get Multiple Jobs

The HTTP method to be used is:
URI will be /task
Parameters: state, group, nodeid, fields






Get Bucket Data

The HTTP method to be used is:
URI will be /bucket
Parameters: bucket, object




Run SQL to get information

The HTTP method to be used is:
URI will be /runsql
Parameters: SQL request (SELECT or SHOW)
