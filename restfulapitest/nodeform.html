Node Management via API

Get All Node States

The HTTP method to be used is:
URI will be /nodestate
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields



Get Node States for System Type

The HTTP method to be used is:
URI will be /nodestate/<systemtype>
Parameters: fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields




Get all Nodes for a System

The HTTP method to be used is:
URI will be /system/<systemid>/node
Parameters: state, state
Optional - limit results to nodes that are in the specified state.   fields fields
Comma separated list of the field names that you would like to be returned. If not specified, all fields are returned.
Default: All fields





Create a Node

The HTTP method to be used is:
URI will be /system/<systemid>/node
Parameters: name, name
The name you want to give to the node (text).   hostname, hostname
The hostname for the node (text).   publicip, publicip
The public IP address of the node in the usual form, e.g. 12.34.56.78   privateip, privateip
The private IP address of the node in the usual form, e.g. 23.45.67.89   port, port
The port number used to access the database on this node - empty or zero if the default is to be used.   instanceid, instanceid
The ID number for the instance (integer).   dbusername, dbusername
The user name for the node database for general queries, process list etc. (text).   dbpassword dbpassword
The password for the node for general queries, process list etc. (text).   repusername, repusername
The user name for the node database replication user (text).   reppassword, reppassword
The password for the node database replication user (text).   scriptrelease, scriptrelease
The current release number for the scripts on the node.   dbtype, dbpassword
The password for the node for general queries, process list etc. (text).   dbversion, repusername
Version of the database system installed on the node.   linuxname reppassword
Name of the Linux distribution installed on the node.   linuxversion reppassword
Version of the Linux distribution installed on the node
